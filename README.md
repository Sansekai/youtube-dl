## Install:
```bash
> pkg update && pkg upgrade -y
> pkg install git
> pkg install ffmpeg
> pkg install nodejs
> git clone https://github.com/sansekai/youtube-dl
> cd yotube-dl
> npm install
```

### Run
```bash
> node index.js
```
